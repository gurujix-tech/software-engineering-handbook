# 🎤 Interview Prep

> Part of the [Software Engineering Handbook](../README.md).

A standalone interview-prep reference, separate from the chapters themselves. The handbook chapters teach concepts from first principles; this section exists purely to help you *prepare for interviews* — every question here comes with a full, interview-ready answer, not just a prompt to test recall.

Organized by topic, and within each topic, by question type:

- **Core Concepts** — definitions, mechanisms, "explain how X works" questions.
- **Scenario-Based** — judgment questions: how you'd handle a real conversation or disagreement.
- **Troubleshooting** — debugging questions: how you'd triage a real production problem.
- **Production** — operating the technology safely and reliably in production: security, resilience, secrets, observability.

New topics are added here only once the corresponding handbook chapters exist.
