# 🖥️ Computer Systems

> Hardware, from the ground up — part of [Computer Science Fundamentals](../README.md) in the [Software Engineering Handbook](../../README.md).

This section covers the physical machine underneath every piece of software you'll ever write or operate: how the CPU, GPU, memory/storage, and motherboard work together, and how peripherals get information in and out. Everything in later chapters — Operating Systems, Linux, Docker, Kubernetes — is ultimately running on top of what's explained here.

Modules are added here only once their content is actually written and reviewed — not upfront as empty placeholders. This index grows alongside the content.
