# 🧠 Computer Science Fundamentals

> The layer everything else in this handbook sits on top of — part of the [Software Engineering Handbook](../README.md).

This section covers the "why does any of this work at all" layer beneath Linux, networking, Docker, and Kubernetes: how a computer actually executes instructions, what an operating system is really managing, how bits become bytes become data on the wire, and what virtualization and the cloud are actually virtualizing. Everything in `linux/`, `docker/`, and `kubernetes/` is an implementation of concepts introduced here — Linux is one OS, not the concept of an OS; a container is one virtualization technique, not the concept of virtualization.

Modules are added here only once their content is actually written and reviewed — not upfront as empty placeholders. This index grows alongside the content.
